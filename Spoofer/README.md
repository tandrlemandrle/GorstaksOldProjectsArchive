# 🎭 Spoofer

> **Anti-Spoofing Detection** - Detects spoofing attempts including MAC, IP, and ARP spoofing.

---

## 📋 Overview

Spoofer detects various spoofing attacks including MAC address spoofing, IP spoofing, and ARP spoofing attempts.

---

## 🎯 What It Detects

- 🎭 **MAC Spoofing** - MAC address changes
- 🌐 **IP Spoofing** - IP address spoofing
- 🔗 **ARP Spoofing** - ARP cache poisoning
- 📡 **Identity Spoofing** - False identity claims

---

## 🚀 Usage

```powershell
# Detect spoofing
.\Spoofer.ps1
```
