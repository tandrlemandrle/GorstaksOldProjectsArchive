# 🗑️ Stripper

> **Metadata Stripper** - Removes sensitive metadata from files before sharing.

---

## 📋 Overview

Stripper removes sensitive metadata from documents, images, and other files to prevent information leakage before sharing files externally.

---

## 🎯 What It Does

- 🗑️ **Metadata Removal** - Strip EXIF, properties
- 📄 **Document Cleaning** - Remove author info
- 🖼️ **Image Cleaning** - Strip GPS, camera info
- 🔒 **Privacy Protection** - Prevent data leaks

---

## 🚀 Usage

```powershell
# Strip metadata
.\Stripper.ps1
```
