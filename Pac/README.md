# 📦 Pac

> **Proxy Auto-Configuration Security** - Monitors and protects PAC file configurations.

---

## 📋 Overview

Auto-Configuration (PAC) file for redirecting traffic (ads) to non existing proxy.

---

## 🎯 What It Does

- 📦 **PAC Monitoring** - Watch PAC file changes
- 🌐 **Proxy Control** - Manage auto-proxy settings
- ⚠️ **Modification Detection** - Detect PAC tampering
- 🛡️ **Traffic Protection** - Prevent proxy hijacking
