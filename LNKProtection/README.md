# 🔗 LNKProtection

> **LNK File Protection** - Protects against malicious shortcut (LNK) file attacks.

---

## 📋 Overview

LNKProtection monitors and protects against malicious LNK (shortcut) file attacks including forced execution, path traversal, and command injection via shortcuts.

---

## 🎯 What It Does

- 🔗 **LNK Analysis** - Examine shortcut files
- 🦠 **Malicious LNK** - Detect weaponized shortcuts
- 🛡️ **Execution Control** - Control LNK execution
- 📝 **Path Validation** - Check target paths

---

## 🚀 Usage

```powershell
# Enable LNK protection
.\LNKProtection.ps1
```
