# 🔍 MiniFilterDrivers

> **Minifilter Driver Monitor** - Monitors for suspicious minifilter drivers and filesystem filter abuse.

---

## 📋 Overview

MiniFilterDrivers monitors for suspicious minifilter driver installations that could be used to intercept filesystem operations, hide files, or perform other malicious activities.

---

## 🎯 What It Detects

- 🔍 **Minifilter Registration** - Filter driver installations
- ⚠️ **Unsigned Filters** - Unverified drivers
- 🗂️ **Suspicious Paths** - Non-standard locations
- 🦠 **Known Bad Drivers** - Malicious filter drivers
