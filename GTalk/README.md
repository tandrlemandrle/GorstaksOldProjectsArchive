# 💬 GTalk

> **Lightweight Python Chat System** - Peer-to-peer messaging with client-server architecture.

---

## 📋 Overview

GTalk is a simple Python-based chat system supporting peer-to-peer communication. It consists of a server for message relay, clients for user interaction, and peer-to-peer capability for direct connections.

---

## 🎯 Features

- 💬 **Text Messaging** - Real-time chat between users
- 🔗 **P2P Support** - Direct peer connections
- 🖥️ **Server Mode** - Central relay option
- 🐍 **Pure Python** - No external dependencies
- 🔒 **Lightweight** - Minimal resource usage

---

## 📁 Project Structure

| File | Description |
|------|-------------|
| `server.py` | Message relay server (1.2 KB) |
| `client.py` | Chat client (1.3 KB) |
| `peer.py` | P2P connection handler (4.6 KB) |

---

## 🚀 Usage

### Start Server
```bash
python server.py
# Server listens on default port
```

### Connect Client
```bash
python client.py
# Connects to server for messaging
```

### P2P Mode
```bash
python peer.py
# Direct peer-to-peer communication
```

---

## 📝 Requirements

- Python 3.6+
- Network connectivity
- Port access for connections
