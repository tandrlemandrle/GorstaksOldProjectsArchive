## 📋 Overview

A collection of custom Rainmeter skins designed by Gorstak for desktop customization and system monitoring. These skins provide visual displays for system information, resource usage, and aesthetic desktop enhancements.

## 🚀 Installation

### Requirements
- [Rainmeter](https://www.rainmeter.net/) installed
- Windows 10/11

### Install Skins
1. **Double-click** any `.rmskin` file
2. **Rainmeter** will open the skin installer
3. **Click Install** to add to your active skins
4. **Configure** position and settings via Rainmeter manager

## ⚙️ Customization

### Skin Settings
- Right-click any skin → **Edit Skin**
- Modify `Variables` section for colors/sizes
- Save and refresh to apply

### Positioning
- Drag skins to reposition
- Use Rainmeter Manage for precise placement
- Save layout for persistence
