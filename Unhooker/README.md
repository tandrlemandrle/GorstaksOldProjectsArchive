# 🔓 Unhooker

> **API Unhooking Detection** - Detects attempts to unhook API functions used by security tools.

---

## 📋 Overview

Unhooker detects API unhooking attempts where malware restores original API functions to bypass security monitoring.

---

## 🎯 What It Detects

- 🔓 **API Unhooking** - Function restoration
- 🎭 **EDR Evasion** - Security bypass
- 🔍 **Hook Detection** - Monitor hooks
- 🚫 **Tamper Detection** - Hook tampering

---

## 🚀 Usage

```powershell
# Detect API unhooking
.\Unhooker.ps1
```
