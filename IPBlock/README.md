# 🚫 IPBlock

> **IP Address Blocking** - Manages IP blocklists and firewall rules for threat IP blocking.

---

## 📋 Overview

IPBlock manages IP address blocklists and automatically applies Windows Firewall rules to block known malicious IP addresses.

---

## 🎯 What It Does

- 🚫 **IP Blocking** - Block malicious IPs
- 📥 **List Management** - Update blocklists
- 🔥 **Firewall Rules** - Auto-create rules
- 🌐 **Geo Blocking** - Country-based blocking

---

## 🚀 Usage

```powershell
# Apply IP blocks
.\IPBlock.ps1
```
